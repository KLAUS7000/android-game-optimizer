## 📦 PASSO 6: Faça Upload do APK

```
┌─────────────────────────────────────────────────────┐
│  Attach binaries by dropping them here or           │
│  selecting them.                                    │
│                                                     │
│  ┌───────────────────────────────────────────────┐ │
│  │                                               │ │
│  │   📂 Clique aqui ou arraste o arquivo       │ │
│  │      app-release.apk                         │ │
│  │                                               │ │
│  │   (Ou simplesmente arraste o arquivo)        │ │
│  │                                               │ │
│  └───────────────────────────────────────────────┘ │
│                                                     │
│  💡 Dica: O arquivo APK está em:                  │
│  app/build/outputs/apk/release/app-release.apk   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Onde Encontrar o APK:
```
seu-pc/
├── android-game-optimizer/
│   ├── app/
│   │   ├── build/
│   │   │   ├── outputs/
│   │   │   │   ├── apk/
│   │   │   │   │   ├── debug/
│   │   │   │   │   │   └── app-debug.apk
│   │   │   │   │   └── release/
│   │   │   │   │       └── app-release.apk 👈 ESTE!
```

### Como Fazer Upload:

**Opção A: Clicar e Selecionar**
1. Clique na área de upload
2. Navegue até: `app/build/outputs/apk/release/`
3. Selecione: `app-release.apk`
4. Clique em "Abrir"

**Opção B: Arrastar e Soltar**
1. Abra seu explorador de arquivos
2. Navegue até: `app/build/outputs/apk/release/`
3. Arraste `app-release.apk` para a área de upload do GitHub

## ❓ Troubleshooting

### ❌ "Erro: Arquivo muito grande"
- GitHub tem limite de 2GB por arquivo
- Seu APK (18MB) está bem dentro do limite ✅

### ❌ "Não consigo encontrar o APK"
- Execute: `./gradlew assembleRelease`
- Procure em: `app/build/outputs/apk/release/app-release.apk`

### ❌ "Publicou errado, como editar?"
- Clique em ⋮ (três pontos) → Edit release
- Faça as mudanças
- Clique em Update release

### ❌ "Como deletar uma release?"
- Clique em ⋮ (três pontos) → Delete
- Confirme
